# Deepscroll

**Open source social media app that replaces doomscrolling with valuable content.**

Instead of mindless scrolling, users discover book summaries, fascinating facts, inspiring people, mind-expanding concepts, thought experiments, and thrilling stories. The algorithm is optimized to keep users engaged as long as possible — but with content that delivers real value.

---

## How It Works

At onboarding, users select their interest areas (e.g. Politics, Technology, Business, Self-improvement, Science, Philosophy, History). The feed adapts automatically — someone interested in Politics sees political book summaries, political facts, political pioneers, and so on. Over time, the algorithm learns which content keeps each user engaged longest.

---

## Feed Categories

| Tab | Description |
|---|---|
| **For You** | Algorithmic mix across all categories |
| **Books** | Book summaries and key takeaways |
| **Facts** | Mind-blowing facts with real depth |
| **People** | Inspiring lives and unsung heroes |
| **Concepts** | Mental models and worldview-expanding ideas |
| **Questions** | Thought experiments and open questions |
| **Stories** | Gripping events that read like a thriller |
